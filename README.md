# Marketing Jur - Site Institucional

## 📋 Visão Geral

Site profissional e institucional para o curso Marketing Jur (www.marketingjur.com.br), desenvolvido com foco em autoridade acadêmica, estratégia empresarial e sofisticação jurídica.

## 🎯 Objetivos do Projeto

- Criar presença digital premium para o curso Marketing Jur
- Estabelecer autoridade no mercado de marketing jurídico
- Converter visitantes em alunos através de design estratégico
- Proporcionar experiência de usuário excepcional

## 🌟 Características Principais

### Design Premium
- **Estética minimalista e institucional**
- **Paleta de cores sofisticada** (azul marinho, dourado, branco)
- **Tipografia elegante** (Inter + Playfair Display)
- **Animações suaves e interações refinadas**

### Funcionalidades Interativas
- **Navegação responsiva com menu mobile**
- **Animações de scroll suaves**
- **Módulos expansíveis** (inspirado no fi.co)
- **Contadores animados de estatísticas**
- **Efeitos hover e microinterações**

### Conteúdo Estruturado
- **Hero section com chamada poderosa**
- **Seção "Para Quem é"** - segmentação clara do público
- **Diferenciais do curso** com ícones e cards
- **Estrutura modular das aulas** - 13 aulas organizadas
- **Footer institucional** com links e informações

## 🎨 Design e Estilo

### Paleta de Cores
- **Primária**: `#1a365d` (Azul marinho profissional)
- **Secundária**: `#d4af37` (Dourado sofisticado)
- **Apoio**: `#2563eb` (Azul vibrante), `#059669` (Verde sucesso)
- **Neutros**: Tons de cinza para textos e backgrounds

### Tipografia
- **Fonte Principal**: Inter (moderna e legível)
- **Fonte Display**: Playfair Display (elegante para títulos)
- **Tamanhos**: Responsivos com clamp() para escalabilidade

### Layout
- **Grid System**: CSS Grid e Flexbox
- **Container**: 1200px máximo, com padding responsivo
- **Mobile-First**: Otimizado para dispositivos móveis

## ⚙️ Funcionalidades Técnicas

### JavaScript
- **Navegação móvel** com menu hambúrguer
- **Animações de scroll** com Intersection Observer
- **Módulos expansíveis** com toggle suave
- **Contadores animados** com requestAnimationFrame
- **Scroll suave** para links internos
- **Navbar inteligente** que aparece/desaparece ao scrollar

### CSS Avançado
- **Variáveis CSS** para consistência
- **Gradientes animados** e sombras profissionais
- **Transições e animações** suaves
- **Media queries** para responsividade total
- **Flexbox e Grid** para layouts modernos

### Responsividade
- **Desktop**: 1024px+
- **Tablet**: 768px - 1024px
- **Mobile**: 480px - 768px
- **Small Mobile**: < 480px

## 🚀 Performance

### Otimizações Implementadas
- **CSS sem bloqueio de renderização**
- **JavaScript carregado após DOMContentLoaded**
- **Animações com requestAnimationFrame**
- **Debouncing e throttling** para eventos de scroll/resize

### Métricas Estimadas
- **Tempo de carregamento**: < 3 segundos
- **First Contentful Paint**: < 1.5 segundos
- **Lighthouse Score**: 90+ (estimado)

## 🔧 Configuração e Uso

### Instalação
1. Clone o repositório ou baixe os arquivos
2. Abra o arquivo `index.html` em um navegador
3. Não requer servidor web para funcionamento básico

### Personalização
- **Cores**: Edite as variáveis CSS em `css/style.css`
- **Conteúdo**: Modifique o HTML em `index.html`
- **Animações**: Ajuste em `js/main.js`

### Deploy
- Hospede em qualquer servidor web estático
- Compatível com GitHub Pages, Netlify, Vercel, etc.
- CDN para assets externos (fonts, ícones)
